# Mini-game
Маленькая игрушка на смартфон, написанная на JavaSkript

Создал для себя. Было интересно повозиться с JS кодом, без использования метода Canvas. Игра всё ещё в процессе создания. Планирую добавить разрушение самалётика при сталкновении с препядствиями, а так же добавить выстрелы при нажатии на кнопку "Fire". Так же адаптировать её под десктопную версию.
Ещё в планах добавить меню, другие препядствия, досчёт очков и по мелочи.
